Duennwald Lab Vector-Digest

If both a DNA sequence and a vector are specified the sequence will be automatically inserted in between the sites specified for all options.

Caution: A small number of Gateway Cloning (pAG) vectors have single base mutations in the att recognition sites, as a result specifying att sites may result in an error (unlikely). If an error occurs your only option is to use any available unique restriction sites that surround the att sites. It does not matter if the restriction site cuts into your sequence.
